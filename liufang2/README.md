# 英语翻译插件

一个浏览器插件，用于将网页上固定的英语文本自动翻译成固定的中文。

## 功能特点

- 🔄 自动翻译网页上的常见英语词汇和短语
- 📝 支持翻译文本内容、按钮、链接、表单占位符等
- 🔍 实时监听页面变化，自动翻译新加载的内容
- ⚡ 轻量级，无需网络请求，本地翻译
- 🎯 精确匹配，只翻译预定义的词汇
- 🎛️ **一键开关控制**，可随时开启或关闭翻译功能
- 💾 开关状态自动保存，刷新页面后仍然有效

## 安装方法

### Chrome/Edge 浏览器

1. 打开浏览器，进入扩展程序管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择本插件的文件夹（包含 manifest.json 的文件夹）

5. 插件安装完成！

### Firefox 浏览器

1. 打开 Firefox，进入 `about:debugging`

2. 点击"此 Firefox"

3. 点击"临时载入附加组件"

4. 选择本插件的 `manifest.json` 文件

## 使用方法

安装插件后：

1. **点击浏览器工具栏中的插件图标**，打开控制面板
2. **使用开关控制插件是否生效**：
   - 开启：插件会自动将页面上的英语文本翻译成中文
   - 关闭：停止翻译功能
3. 开关状态会保存，刷新页面后仍然有效

插件会翻译以下内容：
- 页面文本内容
- 按钮文字
- 链接文字
- 表单占位符（placeholder）
- 图片替代文本（alt）
- 元素标题（title）

## 自定义翻译

如果需要添加或修改翻译，请编辑 `content.js` 文件中的 `translations` 对象：

```javascript
const translations = {
  'English Text': '中文翻译',
  // 添加更多翻译...
};
```

修改后，需要重新加载插件：
1. 进入扩展程序管理页面
2. 找到本插件
3. 点击刷新按钮

## 文件说明

- `manifest.json` - 插件配置文件
- `content.js` - 主要翻译逻辑，包含翻译映射和执行代码
- `popup.html` - 插件控制面板界面
- `popup.js` - 控制面板逻辑，处理开关状态
- `popup.css` - 控制面板样式
- `translations.js` - 翻译映射表（可选，当前已内联到 content.js）
- `README.md` - 本说明文档

## 注意事项

- 插件只翻译预定义的固定词汇，不会翻译整个句子
- 翻译是精确匹配，大小写不敏感
- 不会翻译输入框中的用户输入内容
- 不会翻译代码、脚本等特殊内容

## 支持的翻译类型

插件包含大量常见网页元素的翻译，包括：
- 导航菜单（首页、关于、联系我们等）
- 用户操作（登录、注册、搜索等）
- 表单元素（姓名、邮箱、密码等）
- 购物相关（购物车、结账、订单等）
- 社交媒体（点赞、分享、评论等）
- 状态信息（加载中、错误、成功等）
- 时间相关（今天、昨天、本周等）

## 开发

这是一个 Manifest V3 浏览器插件，使用纯 JavaScript 实现，无需任何依赖。
